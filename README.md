## Scripts Disponibles

Se requiere de Node v18.16.0 para correr el proyecto.

Dentro de la carpeta `backend` y `frontend` se tiene que instalar las dependencias.

### Instalación de dependencias

```bash
npm install
```

### Correr el proyecto en modo de desarrollo

```bash
npm run dev
```
